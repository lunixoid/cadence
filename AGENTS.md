## Версионирование (SemVer)

Проект использует [Semantic Versioning 2.0.0](https://semver.org/lang/ru/): `MAJOR.MINOR.PATCH` (`X.Y.Z`).

| Компонент | Когда увеличивать | Пример |
|-----------|-------------------|--------|
| **X (MAJOR)** | Нарушена обратная совместимость: удалены или изменены публичные API, форматы данных, поведение, от которого могли зависеть пользователи | `1.4.2` → `2.0.0` |
| **Y (MINOR)** | Добавлен или изменён функционал с сохранением обратной совместимости | `1.0.0` → `1.1.0` |
| **Z (PATCH)** | Исправление ошибок без изменения задуманного поведения | `1.1.0` → `1.1.1` |

### Где хранится версия

- **SemVer** — в `MARKETING_VERSION` в [`Cadence.xcodeproj/project.pbxproj`](Cadence.xcodeproj/project.pbxproj) (попадает в `CFBundleShortVersionString`).
- **Build number** — в `CURRENT_PROJECT_VERSION` (целое число, `CFBundleVersion`); увеличивать при каждой сборке/релизе, не смешивать с semver.

Текущая версия: **1.0.0**.

### Обязательное правило для агента

**При первой правке кода в новом чате** (первое изменение файлов проекта в рамках сессии) **обязательно** обновить `MARKETING_VERSION` во всех конфигурациях target'ов в `project.pbxproj` (Cadence и CadenceTests, Debug и Release).

1. Оценить характер изменений по таблице выше и выбрать тип bump: MAJOR, MINOR или PATCH.
2. Обновить все вхождения `MARKETING_VERSION` на новое значение `X.Y.Z`.
3. Кратко указать пользователю, какая версия выставлена и почему.

Если в чате правок кода не было (только обсуждение, план, вопросы) — версию не менять.

### Примеры

| Изменение | Bump | Было → Стало |
|-----------|------|--------------|
| Новый экран настроек, новая интеграция | MINOR | `1.0.0` → `1.1.0` |
| Исправление краша при воспроизведении | PATCH | `1.1.0` → `1.1.1` |
| Удаление поддержки старого формата плейлистов | MAJOR | `1.1.1` → `2.0.0` |
| Рефакторинг без изменения поведения | PATCH | `1.1.0` → `1.1.1` |

При сомнении между MINOR и PATCH — выбирать PATCH только для явных багфиксов; для нового или изменённого пользовательского функционала — MINOR.
